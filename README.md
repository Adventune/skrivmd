## Frontmatter

Options:

| Option | Required | Default         | Info                                        |
| ------ | -------- | --------------- | ------------------------------------------- |
| title  | x        | ""              | Title of the blog post                      |
| author |          | ""              | Will be displayed as the author of the blog |
| date   |          | <last_modified> | Date of publish                             |
| tags   |          | []              | Array of tags to attach to the post         |
